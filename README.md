# E-Commerce Sales Dashboard (Excel)

## Project Overview
This project is an interactive E-Commerce Sales Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, and product trends. The dashboard helps stakeholders monitor key business metrics and make data-driven decisions.

## Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Track monthly sales trends
- Monitor customer and regional performance
- Provide actionable business insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning

## Key Metrics
- Total Sales
- Total Orders
- Average Order Value
- Top Products
- Sales by Category
- Sales by Region
- Monthly Sales Trend

## Dashboard Features
- Interactive filters using slicers
- Dynamic charts and visualizations
- Category-wise sales analysis
- Region-wise performance tracking
- Monthly trend analysis

## Business Insights
- Identified the highest revenue-generating product categories.
- Analyzed seasonal sales patterns.
- Compared regional sales performance.
- Highlighted top-performing products contributing to revenue growth.
